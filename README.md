# titan-url-cli

![demo](demo.png)

titan-url-cli is the terminal client for [titan-url](http://titan-url.herokuapp.com).


## Installation

Via npx:
```
npx titan-url-cli
```

Via npm:
```
npm i -g titan-url-cli
```
